# pyMongoAdmin Documentation

pyMongoAdmin is a simple and pythonic web admin interface for MongoDB. 
With pyMongoAdmin, you can harness the power of python to enter records into
your local MongoDB instance in a comfortable graphical environment.

To get started with pyMongoAdmin, read the [Users' Manual](manual). 
If you have questions, first consult the [FAQ](faq) and then, if that
doesn't help, contact me at zhehao.mao@gmail.com.
